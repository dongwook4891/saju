# FLOW_OOO

## 메인 플로우
1. 사용자가 메인 진입
2. 생년월일/출생일시 입력
3. 제출 클릭
4. 입력값 검증
5. 실패 시 메인에 오류 표시
6. 성공 시 로그인 상태 확인
7. 비로그인: 로그인/회원가입 화면 이동
8. 로그인 성공: 임시 결제 화면 이동
9. 결제완료 클릭: 임시 결과 화면 이동
10. 홈으로 클릭: 메인 이동 및 입력 초기화

## 예외 플로우
- 결제 화면 직접 진입: 로그인 없으면 로그인 화면 이동
- 결과 화면 직접 진입: 로그인 없으면 로그인 화면 이동
- 로그인 취소/실패: 메인 또는 로그인 화면 유지
- 잘못된 입력 형식: 제출 차단
- 결제 화면 홈으로: 메인 초기화
- 결과 화면 홈으로: 메인 초기화
