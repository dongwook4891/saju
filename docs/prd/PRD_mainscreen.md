# PRD_mainscreen

## 목적
비로그인 사용자가 메인화면에서 입력한 생년월일·출생일시를 로그인/회원가입 직후 현재 계정에 저장하고, 로그인 상태로 재방문했을 때 해당 값을 입력값으로 다시 보여준다.

## 범위
- 메인화면 입력값 임시 보관
- 로그인/회원가입 성공 직후 DB 저장
- 메인화면 재진입 시 저장값 조회 및 표시
- 추후 사용자 수정 가능 전제 유지

## 사용자
- 비로그인 상태에서 값을 입력한 뒤 로그인/회원가입하는 사용자
- 기존 저장값이 있는 재방문 로그인 사용자

## 핵심 요구사항
1. 비로그인 입력값은 로그인 전까지 임시 유지된다.
2. 로그인/회원가입 성공 시 현재 입력값을 현재 로그인 계정에 저장한다.
3. 저장값이 있으면 메인화면 input의 실제 값(value)으로 표시한다.
4. 저장값이 없으면 기본 플레이스홀더를 표시한다.
5. 저장·조회는 현재 로그인 계정 기준으로만 수행한다.

## 저장 정책
- 생년월일, 출생 시간, 출생 분이 모두 입력된 경우에만 저장한다.
- 하나라도 비어 있으면 저장하지 않는다.

## 엣지케이스
- 입력값 없이 로그인 성공: 저장 안 함
- 일부만 입력: 저장 안 함
- 저장 실패: 로그인 유지, 사용자 에러 UI 없음, 기본 플레이스홀더 표시
- 다른 계정 로그인: 현재 계정 데이터만 조회
- 첫 회원가입 직후 저장값 없음: 기본 플레이스홀더 표시
