# PRD_resultscreen

## 문서 목적
로그인한 회원이 저장한 사주정보(생년월일, 출생시간)를 바탕으로 AI가 사주 결과를 생성하고, result 화면에서 오늘 / 내일 / 이번 달 / 올해 결과를 확인할 수 있도록 정의한다.  
또한 사용자가 result 화면에서 질문하기 버튼을 눌러, 해당 날짜의 결과를 기준으로 AI와 후속 대화를 이어갈 수 있도록 한다.

## 기능명
회원 사주정보 기반 AI 사주 결과 및 후속 질문 연결 화면

## 기능 목적
- 사용자가 자신의 사주 결과를 한 화면에서 쉽게 확인할 수 있도록 한다.
- 같은 날 재진입 시 동일한 저장 결과를 안정적으로 보여준다.
- 사용자가 결과를 읽은 뒤 별도 채팅 화면에서 추가 질문을 이어갈 수 있도록 한다.

## 주요 사용자
- 로그인 완료 사용자
- 사주정보 입력 및 저장 완료 사용자

## 1차 범위
### 포함
- 저장된 사주정보 조회
- result 화면 진입 시 당일 저장 결과 조회
- 당일 저장 결과가 없으면 Gemini 분석 요청
- Gemini 결과 저장 후 화면 표시
- 오늘 / 내일 / 이번 달 / 올해 결과 표시
- 각 기간별 총평 / 주의점 / 한 줄 조언 표시
- 질문하기 버튼 제공
- 별도 채팅 화면 이동
- 하루 1개 채팅방 유지
- 채팅 화면에서 사용자 질문 입력 및 AI 답변 표시
- 사용자 메시지 / AI 메시지 저장
- 다시 분석하기 버튼 제공
- 다시 분석하기 하루 1회 제한

### 제외
- 성별 기반 해석
- 양력/음력 구분 입력
- 상세 원국 풀이
- 결과 공유
- 푸시 알림
- 다국어 지원
- 음성 입력/출력
- 첨부파일 기반 채팅

## 입력 정보
- 생년월일
- 출생시간

## 입력 규칙
- 생년월일은 필수
- 출생시간은 선택
- 출생시간 미입력 시 시간 미상으로 처리

## 출력 정보
아래 4개 기간 결과를 제공한다.
- 오늘
- 내일
- 이번 달
- 올해

각 기간은 아래 3개 항목으로 고정한다.
- 총평
- 주의점
- 한 줄 조언

## 기간 의미 정의
- 오늘: 기준일 하루의 흐름
- 내일: 기준일 다음 날의 흐름
- 이번 달: 기준일이 포함된 현재 월의 전체 흐름
- 올해: 기준일이 포함된 현재 연도의 전체 흐름

## 핵심 원칙
- 날짜 기준은 Asia/Seoul
- 같은 날에는 저장된 결과를 재사용한다
- 당일 결과가 없을 때만 Gemini를 호출한다
- 사용자가 사주정보를 수정한 뒤 result 화면에 다시 진입하면 새 결과를 생성한다
- 사용자가 직접 누르는 다시 분석하기는 하루 1회만 허용한다
- 초기 생성 실패 후 재시도는 다시 분석하기 횟수에 포함하지 않는다
- 채팅은 해당 날짜의 저장된 result 결과를 기준으로 답변한다
- 채팅방은 사용자 + 날짜 기준 하루 1개만 유지한다

## 사주정보 수정 감지 기준
사주정보 수정 여부는 아래 기준으로 판단한다.
- `profile_updated_at > result_generated_at` 이면 기존 결과를 최신값 기준 결과로 간주하지 않는다
- 이 경우 result 재진입 시 새 결과를 생성한다

## 정상 시나리오
1. 로그인한 사용자가 result 화면에 진입한다.
2. 시스템이 사주정보를 조회한다.
3. 오늘 기준 저장 결과를 확인한다.
4. 저장 결과가 있으면 바로 표시한다.
5. 저장 결과가 없으면 Gemini 분석을 요청한다.
6. Gemini 결과를 검증하고 저장한다.
7. 오늘 / 내일 / 이번 달 / 올해 결과를 카드형으로 표시한다.
8. 사용자는 질문하기 버튼으로 채팅 화면으로 이동할 수 있다.
9. 채팅 화면에서 사용자는 추가 질문을 입력할 수 있다.
10. AI는 해당 날짜의 저장된 사주 결과를 기준으로 답변한다.
11. 메시지는 저장되고, 같은 날 재진입 시 이어서 볼 수 있다.

## 예외 시나리오
- 비로그인 사용자는 로그인 화면으로 이동
- 사주정보가 아예 없으면 입력 화면으로 이동
- 사주정보는 있으나 형식이 잘못되면 수정 화면으로 이동
- Gemini 실패 시 기존 저장 결과 우선 표시, 없으면 재시도 안내
- 응답 형식 오류 시 저장하지 않고 재시도 안내
- 저장 실패 시 오류 안내 및 재시도 제공
- 채팅 화면 진입 시 당일 결과가 없으면 채팅 진입 불가 또는 result 생성 후 진입

## 화면 구성
### 상단
- 페이지 제목
- 기준 날짜
- 저장된 사주정보 요약

### 본문
- 오늘 카드
- 내일 카드
- 이번 달 카드
- 올해 카드

### 하단
- 질문하기 버튼
- 다시 분석하기 버튼

## 채팅 화면 구성
- 채팅방 제목 또는 날짜 표시
- 기존 메시지 목록
- 사용자 입력창
- 전송 버튼

## 연결 기능
- 회원가입/로그인
- 사주정보 입력 및 저장
- 사주정보 수정
- Gemini 분석 요청
- 결과 저장
- 질문하기 버튼을 통한 채팅 화면 이동
- 날짜별 1개 채팅방 유지
- 사용자/AI 메시지 저장

## 성공 기준
- 로그인한 사용자가 result 화면에서 자신의 결과를 확인할 수 있다.
- 같은 날 재진입 시 동일한 결과가 다시 보인다.
- 각 기간에 총평 / 주의점 / 한 줄 조언이 안정적으로 표시된다.
- 질문하기 버튼으로 별도 채팅 화면으로 이동할 수 있다.
- 채팅 화면에서 질문과 답변이 저장되고 같은 날 이어진다.
